## 📖 О проекте

**Yumly** — современное SPA-приложение для создания, хранения и управления кулинарными рецептами.

Проект разработан как портфолио-приложение, демонстрирующее практические навыки современной frontend-разработки: управление состоянием приложения, маршрутизацию, аутентификацию пользователей, CRUD-операции, загрузку изображений и построение масштабируемой архитектуры.

В качестве backend-решения используется **Supabase**, предоставляющий базу данных PostgreSQL, систему аутентификации, файловое хранилище и механизм Row Level Security (RLS).

## 🏗 Архитектура проекта

Проект построен с использованием **feature-based архитектуры**, где логика приложения разделена по бизнес-модулям, а не по типам файлов.

```
src
│
├── app
│   ├── providers
│   └── redux
│
├── assets
├── components
│   ├── navigation
│   └── ui
│
├── context
│
├── features
│   ├── auth
│   └── recipes
│
├── pages
├── supabase
├── utils
```

Такой подход обеспечивает:

- масштабируемость проекта;
- простоту сопровождения;
- повторное использование компонентов;
- разделение ответственности между модулями.

## 🗄 База данных

В качестве базы данных используется **Supabase PostgreSQL**.

### Основные сущности

- Рецепты
- Пользователи (Supabase Auth)
- Изображения (Supabase Storage)

Каждый рецепт содержит:

- название;
- описание;
- изображение;
- кухню;
- уровень сложности;
- время приготовления;
- количество порций;
- ингредиенты;
- пошаговые инструкции;
- теги;
- автора;
- количество лайков.

---

## 🔐 Аутентификация

Аутентификация реализована с помощью **Supabase Auth**.

После входа в систему пользователю становятся доступны:

- создание рецептов;
- редактирование собственных рецептов;
- удаление собственных рецептов;
- управление личной коллекцией.

Для защиты данных используются политики **Row Level Security**, благодаря которым пользователь может изменять только принадлежащие ему рецепты.

## 📦 Установка проекта

Клонируйте репозиторий:

```bash
git clone https://github.com/Maershaa/Yumly-Recipe-App.git
```

Перейдите в папку проекта:

```bash
cd Yumly-Recipe-App
```

Установите зависимости:

```bash
npm install
```

Создайте файл `.env`:

```env
VITE_SUPABASE_URL=your_project_url
VITE_SUPABASE_ANON_KEY=your_anon_key
```

Запустите проект:

```bash
npm run dev
```
